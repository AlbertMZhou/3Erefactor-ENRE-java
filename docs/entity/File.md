## Entity : File

The `.java` files which save the whole java information

### Supported Patterns

```yaml
name: File
```

#### Syntax: Null

```text
While you create a type, first you need to create a file.
```

##### Examples

###### Create A File

```java
//// hello/Foo.java
package hello;

public class Foo{

}

class Bar{

}
```

```yaml
name: File Created
entity:
    type: File
    items:
        -   name: Foo.java
            qualified: hello.Foo.java
            loc: file0
```
